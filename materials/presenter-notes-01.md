# Presenter notes - Session 1: Meet Claude, AI fluency foundations

Instructor-only cues for the student-facing page. The page carries the full six-course content; you present the "Live" cards and run the three demos - "Self-study" cards are homework depth.

## Before the session
- Prepare 5-10 real Q2-style bullets using public or sanitized figures (Demo 1).
- Create a throwaway example Project in advance in case wifi or permissions bite during Demo 2.
- Open the page with all cards collapsed. Present card by card; "Expand all" is the handout view.
- Turn on "Projector zoom: 150%" from the toolbar.

## Timing (45 min)
- 0-3 welcome + the "10% usage" hook
- 3-22 concepts, LIVE cards only, in order:
  - Part 1 (7 min): next-token prediction (run the try-it live), knowledge + Avianca story, steerability (run the 3-audiences try-it live)
  - Part 2 (6 min): first conversation habits, Projects, Artifacts + skills teaser
  - Part 3 (6 min): 4D map + one minute per D; land the description-discernment loop
  - Part 4 (3 min): learning partner + the four techniques
- 22-42 demos: Demo 1 (8 min) → Demo 2 (7 min) → Demo 3 (5 min)
- 42-45 Q&A, close on the cheat sheet + homework

## Demo run-of-show
- Demo 1: run Prompt A on the projector, 20 seconds of silent reading, then Prompt B with your prepared bullets. Cold-call gently on finding the six elements. Do ONE discernment lap live - pick the weakest sentence and fix it with specific feedback. Then 3 minutes of everyone rewriting their own task; two volunteers share.
- Demo 2: build the Project live and narrate every click. The room builds along - pause until most screens show a created Project. If Teams permissions block anything, show yours and let them finish as homework.
- Demo 3: run the teach-me template first with a topic the room votes on (keep it fun), then 3-4 quiet minutes for their own. Debrief: ask who got a quiz answer corrected - name that as the value.

## Cuts if running long
- Cut Part 2 "first conversation" card (most discover it themselves) before cutting anything else.
- Demo 3 can shrink to "start it now, finish as homework".
- Never cut the discernment lap in Demo 1 - it is the single most transferable habit.

## C-level room variant
If the audience skews executive: spend the saved Demo 2 minutes on Part 4 career/learning examples (CMO incrementality, CFO vector database) and frame Projects as "your chief-of-staff briefing book".
